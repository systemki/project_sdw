String 클래스  : ""로 된 문자열 상수에서도 사용 가능
 -charAt(번지) : 문자열에서 번지에 해당하는 문자를 알려주는 메소드
  charAt(0) : 문자열 0번 알려주는 메소드

 - equals(문자열) : 두 문자열이 같은지 알려주는 메소드
		 System.out.println(str1.equals(str2));


 - indexOf(문자열A): 문자열에서 문자열A가 어디에 있는지 번지를
   알려주는 메소드 없으면  -1  출력됨  (찾기) 
	if(str1 != null && str1.indexOf(str2) != -1) { 
					같거나 같지 않거나, -1 

 - length(): 문자열의 길이를 알려주는 메소드 
	System.out.println(str1+"의 길이 : " + str1.length());
					랭스 문자열 길이 출력 	


 -replace(A, B): 문자열에서 A와 일치하는 곳을 B로 교체한 문자열을 
		알려주는 메소드, 원본은 봐뀌지 않음 

	     "제이름은 홍길동입니다.".replace("홍길동","임꺽정"));
		  str3 = str3.replace(str1, str2);
 
 - substring(index) : index번지부터 끝까지 부분 문자열을 생성
 - substring(begin,end):begin번지부터 end번지 전까지
   부분 문자열을 생성 
   기존 문자열 수정하는 것이 아니라 새로운 부분 문자열을 생성
   	// 샘플	String day = num.substring(4,6);
	// 샘플 	String gen = num.substring(7,8);
  	
 - toLowerCase() : 문자열을 전부 소문자로 
 - toUpperCase() : 문자열을 전부 대문자로 
		// System.out.println("Hello" .toLowerCase());	
		// System.out.println("Hello" .toUpperCase());

 - trim() : 앞뒤 공백을 제거하는 메소드
		
		String str = "\n\n\n 안녕하세요  \n\n\n ";
		System.out.println("trim()전 str");
		System.out.println(str);
		System.out.println("trim()후 str");
		System.out.println(str.trim());		


 - valueOf(기본타입) : 기본타입값을 문자열로 변경해준는 메소드 
   클래스 메소드 


 -split(패턴) : 패턴을 기준으로 부분 문자열들을 나눈 후 배열로 만듬

		String str = scan.next();
		String arr[] = str.split(",");
		for(int i = 0; i<str.length(); i++) {
			System.out.println((i+1) + " ." + arr[i]);
		}
