접근제한자 

-public 전체 		: 본인 + 같은패키지 + 자식클래스+다른패키지 > (공개)
-protected	 	: 본인 + 같은패키지 + 자식클래스 (상속에서 다시 설명) > 
-(default) 같은패키지   >  	: 본인 + 같은 패키지 (지인)
-private 본인 클래스 (나) > 	: 본인 (나) 
		      > 본인 클래스의 메스드에서만 사용만 사용 가능 


 클래스가 public이러면 클래스명과  파일명이 같아야 한다. 
 다르면 public을 붙일 수 없음

- 클래스의 멥버 변수 접근방법
 객체명.멤버변수명 

- 일반적으로 대부분 멤버변수는 private 으로 한다.
 >  멤버변수의 값을 직접 수정할 수 없다.
 >  멤버변수의 값을 수정할 수 있는  public 메소드를 만들어줘야한다.
 >  해당 메소드를 getter 와 setter라고 부른다.
     getter 는 privated으로 된 멤버변수의 값을 확인하는 메소드
    setter는 private으로 된 멤버 변수의 값을 설정하는 메소드

- 일반적으로 대부분 멤버메소드는 public으로 한다.


자동차  멤버변수 
제조사,  차종, 바퀴수, 와이퍼, 기어, 자동/수동 등
public  제조사
private : 기어




class A{
	public int num; //public 접근 제한자
	
}

}

	 int num; //디폴트 접근 제한자
