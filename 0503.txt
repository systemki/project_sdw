클래스 
-클래스 선언
- class 클래스명{
	멤버변수들;
	멥버메소드;
	생성자;
}

- 클래스명 작성 규칙은 변수명 작성 규칙과 동일
-  클래스명 작성 관례
-  클래스명의 첫글자는 대문자로 시작 (관례)

 -객체 선언
 클래스명 객체명;
- 객체 생성
	클래스명 객체명 =  new 클래스명 ();
	
- 멤버 변수와 일반 변수의 차임점

	- 일반 변수는 초기값이 자동으로 초기화 되지 않음
	- 멤버 변수는 자동으로 0으로 초기화 됨

- 멤버변수 사용

	객체명: 멤버변수명 = 값;	 

- 생성자 : 멤버 변수들을 초기화 하는곳, 모든 클래는 생성자가 필수 
- 클래스에 생성자가 코드가 없다면 자동으로 기본 생성자가 추가 됨

-기본 생성자 : 클래스에서 생성자가 하나도 없으면 기본 생성자가 자동으로 만들어져서 호출 됨
public 클래스명(){

 }


- 생성자  초기화 작업 
public 클래스명(타입 변수명1, 타입 변수명2,....){
	멤버변수명1 = 변수명1;
	멤버변수명2 = 변수명2;
	....
}


- 메소드 : 기능을 모아놓은 코드 

음료수 자판기 (메소드)
돈, 메뉴, 	      매개변수 (인자)
음료수 	     리턴 타입 


 - 일을 시키려면 제대로된 정보를 전달 > 매개변수 


중국집에서 배달  
배달 		메소드
주소, 메뉴	매개변수(인자) 
요리 		리턴타입


- 메소드 선언 방법
리턴타입 메스드명(매개변수){
	구현;
	return; 
}
			타입 :  
	음료수 음료자판기(int 돈, String 메뉴){
	구현;
}

-리턴타입: 결과물을 묶을 수 있는 타입을 씀 (결과값을 묶음 상태) 음료수로 씀

	
메소드 생성 할때, 다음을 정리
	1. 기능 : 기능을 상세하게 설명  
	2. 매개변수 : 기능을 실행하기 위해 꼭 필요한 정보들을 매개변수로 만듬 (돈,메뉴)
	3. 리턴타입 : 기능을 실행후 알려주는 정보를 리턴타입으로 설정 (음료수)
	4. 메소드명 : 적절한 단어를 이용하여 메소드명을 생성 	

	- 메소드는 상황에 따라서 return 값이 없을수 있다
	- 리턴타입이 없는 경우는 void라는 키워드로 대체
	 메소드의 매개변수는 변수 선언시 값이 복사가 된다.
	매개변수가 참조변수인 경우 원본값이 변경할수있음 (매개변수 →원본값 변경 가능) : 배열, 열거형, 객체 
	매개변수가 일반 변수인 경우 원본 값이 변하지 않음 (원본값 변경 불가능) = 
	일반변수 : 타입이 기본타입경우  값을 저장 	
	참조 변수 : 타입이 기본타입이 아닌경우 , 배열, 열거형 객체, 주소저장  
	-재사용하여 코드 길이 줄이기 위해
	- 기능의 유지보수가 쉬움 
	-코드에 구조를 파악하기 쉬워짐
